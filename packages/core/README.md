# @soniox/core

Core types, errors, and utilities for Soniox SDKs.

This package is platform-agnostic and serves as the foundation for both `@soniox/node` and `@soniox/web` packages.

## Installation

```bash
npm install @soniox/core
```

> **Note:** You typically don't need to install this package directly. It's automatically included when you install `@soniox/node` or `@soniox/web`.
